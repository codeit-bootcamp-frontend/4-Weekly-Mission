### 상태 (Pseudo-classes):

1. **`:active`**: 요소가 활성화된 상태를 나타냅니다. 사용자가 요소를 활성화했을 때 (예: 클릭한 경우) 적용됩니다.
2. **`:hover`**: 요소에 마우스가 올라가 있는 상태를 나타냅니다. 마우스 커서가 요소 위에 있을 때 적용됩니다.
3. **`:focus`**: 요소가 포커스를 받은 상태를 나타냅니다. 보통 키보드 탐색이나 터치로 요소를 선택한 경우 적용됩니다.
4. **`:visited`**: 링크가 방문되었음을 나타냅니다. 사용자가 이미 방문한 링크를 나타낼 때 적용됩니다.
5. **`:focus-within`**: 부모 요소나 자식 요소 중 하나가 포커스를 받았을 때 적용됩니다. 즉, 해당 요소 내에 포커스가 있는지 여부를 나타냅니다.
6. **`:focus-visible`**: 사용자가 키보드로 요소에 포커스를 줄 때만 적용됩니다. 마우스로 요소에 클릭하여 포커스를 준 경우에는 적용되지 않습니다.
7. **`:target`**: URI의 fragment 식별자에 해당하는 요소를 나타냅니다. 예를 들어, `#section`과 같은 URL의 fragment에 해당하는 요소가 있을 때 적용됩니다.

### 이벤트 (Events):

1. **`onClick`**: 요소를 클릭했을 때 발생합니다.
2. **`onMouseDown`**: 요소 위에서 마우스 버튼을 눌렀을 때 발생합니다.
3. **`onMouseUp`**: 요소 위에서 마우스 버튼을 놓았을 때 발생합니다.
4. **`onMouseMove`**: 마우스가 요소 위에서 움직일 때 발생합니다.
5. **`onMouseOver`**: 마우스 커서가 요소 위로 이동할 때 발생합니다.
6. **`onMouseOut`**: 마우스 커서가 요소를 벗어날 때 발생합니다.
7. **`onFocus`**: 요소에 포커스가 들어올 때 발생합니다.
8. **`onBlur`**: 요소에서 포커스가 빠져나갈 때 발생합니다.
9. **`onKeyDown`**: 키보드에서 키를 눌렀을 때 발생합니다.
10. **`onKeyPress`**: 키보드에서 키를 눌렀을 때 발생합니다.
11. **`onKeyUp`**: 키보드에서 키를 놓았을 때 발생합니다.
12. **`onChange`**: 요소의 값이 변경될 때 발생합니다.
13. **`onInput`**: 요소에 입력이 발생할 때 발생합니다. (일부 입력 요소에서는 사용되지 않을 수 있음)
